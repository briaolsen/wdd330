JavaScript Validation
  • Constraint Validation API - use on: button, fieldset, input, output, select, textarea
  • Properties Available: validationMessage, validity, will validate
  • ValidityState - patternMismatch, tooLong, tooShort, rangeOverflow, rangeUnderflow, typeMismatch, valid, valueMissing
  • Methods Available: checkValidity(), setCustomValidity()
Summary
  • Display explicit error messages
  • Be permissive about the input format
  • Point out exactly where the error occurs, especially on large forms
