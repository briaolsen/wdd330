Chapter 5 - Objects

Creating Objects

const spiderman = {};
// empty object const name = 'Iron Man';
const realName = 'Tony Stark';
const ironMan = { name: name, realName: realName }; //long way
const ironMan = { name, realName }; //short ES6 way, property key is the same as a variable name that the property value is assigned to

Accessing Properties

You can use the dot notation or square brackets to access an object's properties:
superman.name or superman['name'] or superman["real" + " " + "name"]

Computed Properties

Can use JavaScript code in square brackets for the property key:
const hulk = { name: 'Hulk', ['catch' + 'Phrase']: 'Hulk Smash!' };
const bewitched = true;
const captainBritain = { name: 'Captain Britain', hero: bewitched ? false : true };

You can also use a symbol as a key:
const realName = Symbol('real name');
supergirl[realName] = 'Kara Danvers';

Other Notes about Object Properties and Methods
  • You can call an object's method with dot or bracket notation. superman.fly() or superman['fly']()
  • Use the in operator to check for a property: 'city' in superman (returns false/true)
  • Also can use hasOwnProperty() method to see if an object has it's own property (not one inherited)
  • Loop through properties using for in loop: for(const key in superman){ console.log(key + ": " + superman[key]);}
  • use Object.keys(objectname) will return an array of all the keys, Object.values(objectname) will return array of all the values, Object.entries(objectname) returns array of key-value pairs (use of to iterate through)
  • Add new properties and methods: superman.city = 'Metropolis'
  • Change values: superman['real name'] = 'Kal-El'
  • Remove properties: delete superman.fly
  • Nested Objects: Objects can contain objects
  • Objects are copied by reference (change one and change the other copy)
Objects as Parameters to Functions
  • An object can be passed as a parameter to a function
  • The parameters can be in different order and can be assigned default values
Namespacing

Use an object to create a namespace for a group of functions to avoid colliding with other functions of the same name.

JSON
  1. Property names must be double-quoted
  2. Permitted values are double-quoted strings, numbers, true, false, null, arrays and objects
  3. Functions are not permitted values
  • JSON.parse() converts JSON data to a JS object
  • JSON.stringify() converts JS object to JSON data
MATH Object
  • Properties: Math.PI, Math.SQRT2, etc.
  • Methods: Math.abs(), Math.ceil(), Math.floor(), Math.log(), Math.max(), Math.sin(), Math.random(), etc
  • Rounding creates some issues and errors, keep it in mind
DATE Object
  • Constructor: new Date(); (defaults to current date) or set it by ('year,month,day,hour,minutes,seconds,milliseconds')
  • Methods: Math.abs(), Math.ceil(), Math.floor(), Math.log(), Math.max(), Math.sin(), Math.random(), etc
  • Rounding creates some issues and errors, keep it in mind

Chatper 6 - The Document Object Model

Node Lists
  • document.images (etc.) -> returns a node list of all the elements
  • Accessed using [ ] square brackets and .length with for loop
  • Not an array but can be turned into an array const imageArray = Array.from(document.images)
  • getElementById, getElementsByTagName, getElementsByClassName
  • querySelector -> ('#bats') ('.hero') gives first one, querySelectorAll gives a node list
  • Example: const ul = document.querySelector('ul#roster'); const batman = ul.querySelector('li#bats');
  • Can use .children .firstChild .lastChild .length .parentNode .nextSibling .previousSibling etc. on node lists
  • .nodeValue .textContent (innerText)
  • .id .className - can be used to find or change
  • .classList -> use .add .remove .toggle .contains
