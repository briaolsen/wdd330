Chapter 2 - Programming Basics

var vs let vs const

let and const have block scope, preventing you from overwriting any built-in methods by assignment
block scope - values only exist inside the block they are declared in
global scope - declared outside of a block and is accessible anywhere in the program
local scope - any variables defined inside a block using let or const are only available inside that block (otherwise they are global)

Template Literals

Template literals use the backtick character ` to deliminate the string. Can use both types of quote marks within. Also allow insertion of JavaScript code with ${ }.

Chapter 3 - Arrays, Logic, and Loops

Array Methods

array.length -> find the length of an array
array.pop() -> returns the last item of the array
array.shift() -> removes the first item in the array
array.push('item') -> appends a new value to the end of the array, returns new length
array.unshift('item') -> appends a new item to the beginning of the array, returns new length
array.concat(['item1', 'item2']) -> merge an array with one or more arrays and returns the new array
array.join(' & ') -> returns a string comprising of all the items in the array separated by commas by default, or you can choose a separator
array.slice(2, 4) -> returns a subarray, chopping out a slice of the original array starting with first number, not including second number
array.splice(3, 1, 'New Item') -> removes items from the array and inserts new items in their place, (start, #removed, inserted), returns the items removed, can remove 0 items
array.reverse() -> permanently reverses the order of the array
array.sort() -> permanently sorts the array by alphabetical order
array.indexOf('item') -> returns the index, or -1 if not included
array.includes('item', 1) -> returns a boolean if element is in the array, can also specify where to start the search
... spread operator -> three dots placed in front of an array, spreads out the elements of the array, can be used to spread out two arrays and merge, can also be used to flatten multidimensional arrays

Ternary Operator

condition ? (//code to run if condition is true) : (//code to run if condition is false)

For-Of Loop

ARRAYS: for(const value of array) { do something with value; } -> value represents the value of each element in the array
MAPS: for(const key of map.keys()) { do something with key } or for(const value of map.values()) { do something with value }

Chapter Summary

  1. Arrays are an ordered list of values
  2. Multidimensional arrays are arrays that contain other arrays
  3. Arrays have lots of methods that can be used to manipulate items in the array
  4. Sets are new in ES6 and are ordered lists of non-duplicate values
  5. Maps are new in ES6 and are ordered lists of key-value pairs
  6. We can use an if and else statement to control the flow of code
  7. The switch statement can be used instead of multiple if and else statements
  8. A while loop and do ... while loop can be used to repeat a block of code while a condition is still true
  9. A for loop works in a similar way to a while loop, but has a different syntax
  10. A for-of loop can be used to iterate over an array
  11. Sets and maps are enumerable, so can also be looped over using a for-of loop

Chapter 4 - Functions

Named Function Literal

function hello(){ console.log('Hello World');}

Anonymous Function Expression

const goodbye = function(){ console.log('Goodbye World!');};

Named Function Expression

const goodbye = function bye(){ console.log('Goodbye World!');};

Arguments

Can use arguments() function to see the arguments. It is not an array so you can't use array methods.

Rest operator: creates an array of arguments available inside the body of the function
function rest(...args){ return args; } -> args is an array
To use rest operator put three dots in front of the last parameter in a function declaration. This collects all the args entered into an array. Can use a for-loop to get to them all.

Default Parameters

Default parameters -> values that will be used if no argument is specified.
function hello(name='World') { }
Put default at the end of the arguments.

Arrow Functions

Parameters come before the arrow and the main body comes after. Always anonymous and must be assigned to a variable.
const square = x => x*x;
const add = (x,y) => x + y; const hello = () => alert('Hello World!'); Single parameters don't need () Body not in {} if only one line. Don't need return if one line. Don't bind their own value of this to the function.

Arrow functions aren't great to use for longer functions:
const tax = (salary) => {
const taxable = salary - 8000;
const lowerRate = 0.25 * taxable;
taxable = taxable - 20000;
const higherRate = 0.4 * taxable;
return lowerRate + higherRate;
}

Hoisting

Move all variable and function declarations to the top of the current scope so they can be used whenever. Best used for functions and not variables and not for function expressions (anonymous function is assigned to a variable).

Callbacks

Functions that are passed as an argument to another.

function sing(song, callback) {
console.log(`I'm singing along to ${song}`.);
callback();
}

function dance() {
console.log(`Dancing`);
}

sing('Let It Go', dance);
sing('Let It Go',()=>{ console.log("I'm standing on my head.");});

For-Each Function

forEach() takes three parameters - value in the array, index, the array it's being called on

colors.forEach( (color, index) =>
console.log(`Color at position ${index} is ${color}`) );

Map Function

Iterates over an array and takes a callback function as a parameter. It returns a new array that replaces each value with the return value of the callback function.

[1,2,3].map( x => 2 * x);
<< [2,4,6]

Other Methods for Arrays

Reduce() - Iterates over each value in the array and combines each result to return just a single value.
Filter() - returns a new array that only contains items from the original array that return true when passed to a callback

Chaining Iterators

You can use an iterator function on an array returned by an iterator function.
